# Paper Reading Skills

Codex skills for paper discovery, reading, and quality triage.

## Available Skills

- `paper-quality`: score deep learning, robotics, and computer vision papers on a `5..0` scale, where `5` is must-read/seminal and `0` is not sure or insufficiently verified.

## Install

```bash
python3 /path/to/install-skill-from-github.py --repo HLrwth/paper_reading_skills --path skills/paper-quality
```

Or install with Codex's skill installer from:

`https://github.com/HLrwth/paper_reading_skills/tree/main/skills/paper-quality`

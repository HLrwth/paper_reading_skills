# Paper Reading Skills

Codex skills for paper discovery, reading, summary, and quality triage.

## Available Skills

- `paper-quality`: score deep learning, robotics, and computer vision papers on a `5..0` scale, where `5` is must-read/seminal and `0` is not sure or insufficiently verified.
- `paper-summary`: write self-contained deep paper summaries with motivation, method, core equations, training setup, experiments, limitations, and connections to related methods.

## Install

Install both skills:

```bash
python3 /path/to/install-skill-from-github.py --repo HLrwth/paper_reading_skills --path skills/paper-quality skills/paper-summary
```

Install one skill:

```bash
python3 /path/to/install-skill-from-github.py --repo HLrwth/paper_reading_skills --path skills/paper-summary
```

GitHub paths:

- `https://github.com/HLrwth/paper_reading_skills/tree/main/skills/paper-quality`
- `https://github.com/HLrwth/paper_reading_skills/tree/main/skills/paper-summary`
